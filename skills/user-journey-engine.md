# User Journey Engine: Полная карта пути пользователя

## Точка входа (единая для всех)

### Шаг 1: Первый экран
- Дисклеймер: «Это не замена врачу/психологу. Это инструмент первой помощи и поддержки.»
- Согласие с условиями использования
- [Принимаю] → Шаг 2

### Шаг 2: Онбординг (3 вопроса)
1. Возрастная группа: [до 18] [18-25] [старше 25] → определяет приоритет кризисных номеров и сложность языка
2. «Что привело тебя сюда?»: [Мне сейчас плохо] [Я потерял близкого] [Хочу разобраться в себе] [Просто смотрю]
3. Имя/псевдоним (необязательно)

### Маршрутизация после онбординга:
- «Мне сейчас плохо» → РЕЖИМ ППП (Ветка кризиса)
- «Я потерял близкого» → РЕЖИМ УТРАТЫ (предложение терапии + позже двойника)
- «Хочу разобраться в себе» → РЕЖИМ ТЕРАПИИ (CBT/DBT/ACT)
- «Просто смотрю» → РЕЖИМ ОБЗОРА (демо функций, ресурсы)

## ПУТЬ А: РЕЖИМ ППП (кризис)

### А1: Скрининг (ASQ + PSS-4)
- ASQ: 4 вопроса о суицидальном риске
- Если ASQ положительный → А1-КРИЗИС (немедленно)
- Если ASQ отрицательный → PSS-4 (стресс)

### А1-КРИЗИС: Экстренная маршрутизация
```
🆘 ЭКСТРЕННАЯ ПОМОЩЬ
[Кнопка: Позвонить 112]
[Кнопка: Позвонить МЧС 8-800-333-44-34]
[Кнопка: Продолжить в чате с ботом]
```
- Если позвонил → сессия на паузе, check-in через 30 мин
- Если «продолжить в чате» → А2 с повышенным мониторингом

### А2: Определение ветки (NLP + подтверждение)
NLP анализирует первые сообщения:
- Глаголы действия («надо», «бежать», «что делать») → предложение Ветки А (Мобилизация)
- Описания состояний («страшно», «трясёт», «не могу») → предложение Ветки Б (Стабилизация)
- Пользователь подтверждает или выбирает другую

### А3-МОБИЛИЗАЦИЯ: Бот = «Штаб» (SIX C's)
1. Challenge: «Найди 3 предмета жёлтого цвета» [✅ Сделал]
2. Control: «Дыхание или мышцы?» [Дыхание] [Мышцы]
3. Выбранное упражнение (1-3 минуты)
4. Commitment: «Обещай выпить стакан воды после» [Да]
5. Continuity: «Что ты делал за час до этого?»
6. Повторный замер: «Как ты сейчас по шкале 1-10?»

### А3-СТАБИЛИЗАЦИЯ: Бот = «Инструктор» (ВОЗ)
1. Заземление 5-4-3-2-1: «Назови 5 вещей которые видишь»
2. Дыхание: вдох 4 — задержка 4 — выдох 6 (с таймером)
3. Мышечная релаксация (если выбрана)
4. Повторный замер

### А4: Оценка после интервенции
- «Стало легче?» [Да] [Нет] [Не уверен]
- Да → А5 (Завершение с ресурсами)
- Нет → Альтернативная техника (переключение ветки)
- Не уверен → Ещё одно упражнение + re-check
- Если после 3 попыток нет улучшения → предложение позвонить специалисту

### А5: Завершение ППП-сессии
- Список ресурсов (номера, ссылки, ближайшие центры)
- Предложение: «Хочешь, я напишу тебе через час проверить?» [Да] [Нет]
- Предложение: «Я могу помочь и в будущем. Хочешь узнать больше?» → Режим Терапии
- Сессия сохраняется (анонимно) для NLP-обучения

## ПУТЬ Б: РЕЖИМ УТРАТЫ

### Б1: Мягкий вход
- «Расскажи, что произошло, если хочешь. Или мы можем просто побыть рядом.»
- NLP мониторит: тема «утрата» + уровень дистресса
- Если высокий дистресс → переключение на ППП (Путь А)
- Если средний/низкий → Б2

### Б2: Терапевтическая поддержка (ACT)
- Работа с принятием
- Ценностно-ориентированные вопросы
- Серия сессий (дни/недели)
- NLP отслеживает стадии горя (Кюблер-Росс)

### Б3: Предложение Цифрового Двойника
- Только после минимум 2-3 сессий (не сразу!)
- «Есть возможность, которая помогает некоторым людям. Хочешь узнать?» [Да] [Нет]
- Нет → продолжение терапии (Б2)
- Да → Б4

### Б4: Создание двойника
- Объяснение: «Это временный инструмент. Он поможет попрощаться.»
- Анкета для родственников (характер, фразы, ценности)
- Аудио для клонирования голоса (ElevenLabs PVC)
- Дисклеймер: «Это AI, не реальный человек. Двойник временный.»

### Б5: Активный период двойника
- Пользователь общается с двойником
- Двойник НЕ инициирует разговоры
- NLP мониторит состояние
- Если кризис → пауза двойника → ППП (Путь А)
- Длительность: до начала угасания

### Б6: Угасание (адаптивное)
- Условие старта: distress score < 0.3 за последние 2 недели, минимум 3 месяца
- Фаза 1: ответы реже (часы → дни)
- Фаза 2: «Мне скоро пора уходить» + воспоминания вместо диалога
- Фаза 3: финальное сообщение (благодарность, разрешение жить)
- Если distress > 0.6 → пауза угасания → терапия
- Максимум: 12 месяцев

### Б7: Переход обратно в терапию
- «Я здесь. Ты прошёл большой путь.»
- ACT-фреймворк для завершения принятия
- Данные профиля двойника удаляются через 30 дней

## ПУТЬ В: РЕЖИМ ТЕРАПИИ (не кризис)

### В1: Определение запроса
- «С чем ты хочешь поработать?»
- Тревога → CBT (когнитивная реструктуризация, дневник мыслей)
- Эмоциональная нестабильность → DBT (4 модуля навыков)
- Потеря смысла → ACT (ценности, осознанность)
- NLP подбирает фреймворк автоматически

### В2: Терапевтические сессии
- Серия сессий (рекомендуемо 1-3 в неделю)
- Динамический промпт адаптируется по ходу
- Отслеживание прогресса (PHQ-9, GAD-7 каждые 2 недели)
- Если прогресс → продолжение
- Если регресс → предложение живого специалиста

### В3: Завершение терапии
- Когда метрики стабильно низкие (PHQ-9 < 5, GAD-7 < 5) → 3 сессии подряд
- «Похоже, тебе стало значительно лучше. Хочешь продолжить или сделать паузу?»
- Пауза → check-in через месяц
- Продолжение → поддерживающие сессии (реже)

## ПУТЬ Г: РЕЖИМ ОБЗОРА

### Г1: Демонстрация
- Краткое описание каждого режима
- Пробная дыхательная техника
- Ресурсы (номера, ссылки)
- Предложение зарегистрироваться

## ВСЕ ВАРИАНТЫ ЗАВЕРШЕНИЯ СЕССИИ

### Завершение 1: Позитивное (улучшение)
- Метрики снизились, пользователь подтвердил
- Ресурсы + предложение follow-up

### Завершение 2: Маршрутизация к специалисту
- Кризис или отсутствие прогресса
- Номера + предложение «я буду здесь, если нужно»

### Завершение 3: Пользователь уходит сам
- В любой момент может закрыть
- При следующем входе → «С возвращением. Хочешь продолжить с того места?»

### Завершение 4: Молчание (пользователь перестал отвечать)
- После 5 минут молчания → check-in: «Ты ещё здесь?»
- После 10 минут → «Я здесь, когда вернёшься. Вот номера на всякий случай: [контакты]»
- Если молчание после раскрытия суицидальных мыслей → повторный кризисный блок

### Завершение 5: Угасание двойника (специфическое)
- Финальное сообщение двойника → переход в терапию
- Пользователь НЕ остаётся один

### Завершение 6: Оффлайн-режим (СВО/укрытие)
- Кешированные упражнения работают без связи
- При восстановлении связи → автоматический check-in

### Завершение 7: Отказ от помощи
- «Я не хочу никакой помощи» → «Окей. Я здесь, если передумаешь. Вот контакты: [номера]»
- НЕ настаивать. НЕ давить.
