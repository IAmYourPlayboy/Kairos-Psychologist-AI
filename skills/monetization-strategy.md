# Monetization Strategy

## Этический фундамент
Кризисная помощь (ППП-бот) — ВСЕГДА бесплатна. Человек в панике не должен видеть paywall. Это не обсуждается.

## Модель: Freemium с этическим ядром

### Бесплатный уровень (навсегда)
- ППП-бот: скрининг, SIX C's, ВОЗ протоколы, кризисная маршрутизация
- Базовые дыхательные и заземляющие упражнения
- Список верифицированных ресурсов
- Оффлайн-режим (кешированные упражнения)
- Ограничение: 3 терапевтические сессии в месяц

### Подписка «Поддержка» (~500-900 руб/мес)
- Безлимитные терапевтические сессии (CBT/DBT/ACT)
- Отслеживание прогресса (PHQ-9, GAD-7 динамика)
- Персонализированные упражнения
- Follow-up напоминания
- Голосовой режим (ElevenLabs TTS для инструкций)

### Подписка «Двойник» (~1500-3000 руб/мес)
- Всё из «Поддержки»
- Создание цифрового двойника (один профиль)
- Клонирование голоса (ElevenLabs PVC)
- Полный цикл угасания (6-12 месяцев)
- Переход в терапию после завершения
- Высокая стоимость обоснована: PVC стоит денег, GPU для инференса

### B2B направление
- Корпоративные лицензии (HR: поддержка сотрудников) — фикс за сотрудника/месяц
- Интеграция с клиниками (скрининг → направление к специалисту) — партнёрская комиссия
- Белая этикетка для НКО, МЧС, военных — адаптация под специфику

### Гранты
- Фонд президентских грантов (социальные проекты РФ)
- «Фонд содействия инновациям» (УМНИК: до 500K руб, СТАРТ: до 2M руб)
- Международные: Wellcome Trust, Grand Challenges
- Региональные IT-гранты (Сколково, РВК)

## Юнит-экономика (ориентир)
- Стоимость GPU инференса (Qwen 2.5 72B): ~$0.02-0.05 за сессию
- ElevenLabs TTS: ~$0.01-0.03 за сообщение
- ElevenLabs PVC: $5/мес за голос (план Creator)
- Сервер в РФ (GPU): ~$500-1500/мес (зависит от нагрузки)
- При 1000 платных пользователей «Поддержки» (700 руб/мес) = 700K руб/мес → покрывает инфраструктуру

## Рекомендация
Начать с гранта → бесплатная версия → набрать 1000+ пользователей → включить платные тарифы. Кризисная помощь = маркетинговый канал (люди приходят бесплатно → остаются на платные сервисы).
