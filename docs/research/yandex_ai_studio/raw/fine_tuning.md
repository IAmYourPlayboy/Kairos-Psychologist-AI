## Обзор

Yandex AI Studio позволяет дообучать по методу **LoRA** (Low-Rank Adaptation of Large Language Models):
- модель генерации текста **YandexGPT Lite**;
- классификаторы на базе **YandexGPT Lite**;
- модель эмбеддингов.

> ⚠️ Функциональность находится на стадии **Preview**.

> ⚠️⚠️ **КРИТИЧНО ДЛЯ НАШЕГО ПРОЕКТА**: дообучение **НЕ доступно** для Qwen, DeepSeek, gpt-oss, Gemma. Только YandexGPT Lite + классификаторы + эмбеддинги.

---

## Возможности дообучения моделей генерации текста

> Модели генерации текста **не получится дообучить новой информации**, например базе знаний службы поддержки. Однако вы можете обучить модель отдавать ответ в определённом формате или анализировать текст.

Что можно через LoRA:
- пересказывать и изменять формулировки текстов;
- генерировать вопросы к тексту и ответы на них;
- форматировать ответы в определённой стилистике или формате;
- классифицировать тексты, обращения и диалоги;
- извлекать сущности из текста;
- дообучать модели классификации и эмбеддингов.

## Процесс дообучения

1. Подготовить данные (см. разделы «Датасеты для генерации текста», «Датасеты для классификации текста», «Датасеты для эмбеддингов» в документации).
2. Загрузить датасет в Yandex Cloud (по умолчанию **до 5 ГБ** на один датасет).
3. Запустить дообучение, указав тип и (опционально) параметры.
4. Дообучение займёт **от 1 часа до 1 суток** в зависимости от объёма данных и загруженности системы.

Для дообучения нужна роль **`ai.editor`**.

## Запросы к дообученным моделям

После завершения обучения вы получите идентификатор дообученной модели. Этот идентификатор передаётся в поле `modelUri` тела запроса.

- Обращаться к дообученной модели генерации текста — через **API генерации текста**, из Yandex DataSphere и других приложений.
- К дообученному классификатору — через метод `classify` Text Classification API.
- Можно использовать **Yandex AI Studio SDK** для работы с дообученными моделями.

> ⚠️ Для корректной работы дообученной модели **указывайте инструкцию, использованную при обучении**, в сообщении с ролью отправителя `system`.

Для отправки запросов через API в ноутбуках DataSphere — добавить пользовательский или сервисный аккаунт в список участников проекта DataSphere с ролью **`ai.languageModels.user`**.

## Тарификация дообучения

> На стадии Preview процесс дообучения моделей **не тарифицируется**.
> Дообученная модель YandexGPT Lite тарифицируется **как базовая модель YandexGPT Lite** (то есть 0,2 ₽ / 1К токенов в синхронном режиме).
