## Контекст

Тех.поддержка Yandex Cloud сообщила, что под наш аккаунт доступны
**только два варианта** Qwen:

1. **Qwen3.6 35B-A3B** — `gpt://<folder>/qwen3.6-35b-a3b`
2. **Qwen3 235B-A22B-FP8** — `gpt://<folder>/qwen3-235b-a22b-fp8`

Оба:
- 256K контекстного окна (262144 токена)
- OpenAI-совместимый API через `/v1/chat/completions`

Задача: на основании реальных данных с сайта (цены, лимиты, бенчмарки,
условия) принять решение **какую из двух подключать**.
